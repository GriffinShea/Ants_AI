These are instructions for the OpenGL setup in this project to be used in a new Visual Studio project.

In fresh VS project folder (with .sln file) copy glad.c and Debug/freeglut.dll. Also you must create the folder saved_generations.

Copy Libraries to the folder with the .vcxproj file.

In project>properties switch to Debug and Win32 at the top (Configuration).

In project>properties:
	> linker > input > additional dependencies
		--> change to "glew32s.lib;SOIL.lib;freeglut.lib;%(AdditionalDependencies)"

	> configuration properties > vc++ directories > include directories
		--> change to "$(ProjectDir)/Libraries/include;$(IncludePath)"

	> configuration properties > vc++ directories > library directories
		--> change to "$(ProjectDir)/Libraries/lib;$(LibraryPath)"
